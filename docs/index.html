The Problem

AI agents are making thousands of decisions per day — querying databases, calling APIs, transforming data, optimizing parameters, writing code. But there's no audit trail. No way to prove what happened, when, or why.

For regulated industries, scientific research, and any team that needs accountability — this is a blocker. QuantaOptima fixes it.

How the Audit Chain Works

Action 1: "query" Action 2: "decide" Action 3: "execute" ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐ │ before: {input} │ │ before: {options} │ │ before: {plan} │ │ after: {result} │ │ after: {choice} │ │ after: {output} │ │ sig: HMAC-SHA256( │──chain──│ sig: HMAC-SHA256( │──chain──│ sig: HMAC-SHA256( │ │ prev_sig + data│ │ prev_sig + data│ │ prev_sig + data│ │ ) │ │ ) │ │ ) │ └──────────────────┘ └──────────────────┘ └──────────────────┘ Tamper with Block 1 → Blocks 2 and 3 signatures become INVALID

Same principle as blockchain, applied to AI agent actions. Every block's signature depends on the previous block's signature. Change anything, anywhere, and verification fails from that point forward.

Two Ways to Use QuantaOptima

As an MCP Server

Your AI agent can log actions, verify the chain, and export the audit trail through natural language. 10 tools available out of the box. Setup takes 30 seconds.

{
  "mcpServers": {

As a Python Library

Embed the audit chain in your own MCP server or application. Use the @auditable decorator to auto-log any function call.

from quantaoptima import AuditChain

chain = AuditChain(scope="my-server")
chain.log("query", {"q": "..."}, {"result": "..."})
assert chain.verify()

Honest Positioning

The built-in optimizer is a demo of the audit chain in action, not a replacement for scipy. On raw solution quality, scipy's Dual Annealing finds better answers at similar evaluation budgets. But scipy can't talk to your AI agent, can't prove its work cryptographically, and can't show you how it explored the landscape. Different tools for different problems.

Use QuantaOptima when you need auditable, interpretable AI agent actions. Use scipy when you only need a number.
